E-commerce Sales & Customer Analytics (SQL)
Project Overview
This project analyzes an e-commerce dataset using SQL to understand sales performance, customer behavior, and product profitability.

The goal is to demonstrate real-world SQL skills used by data analysts.

Key Objectives
Analyze total and monthly revenue
Identify repeat vs new customers
Calculate customer lifetime value (CLV)
Find top-performing and most profitable products
Tools Used
SQL (PostgreSQL)
